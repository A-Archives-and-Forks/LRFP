### Plugins

To distinguish between implementations based on the rooting and framework layers, we will refer to implementations (e.g., LSPosed) based on the rooting layer as modules, 
implementations (e.g., HMA) based on framework layers as plugins, and scripts (e.g., scripts written in Java) written based on plugins as scripts. 

Since each framework layer manager has a download tab and the plugin repository is already quite mature, no plugins are stored here. 
Please download and install them from the plugin repository or Telegram channel if necessary. 

---

### 插件

为区别基于 root 层和框架层的实现，我们将基于 root 层的实现（如 LSPosed）称为模块，
基于框架层的实现（如隐藏应用列表）称为插件，将基于插件编写的脚本（如 Java 脚本）称为脚本。

由于各框架层管理器均有下载选项卡且插件仓库已十分成熟，此处不存储任何插件。
如有需要，请从插件仓库或电报频道下载安装。

## Reasons

This document will briefly discuss some reasons for conducting the LRFP. 

### Why Rooting

Although the era of everybody rooting and one-click rooting is over, some users still root their Android phones before using them. 
Currently, most people believe that the Android system is mature enough and there is no need to root it. 
However, nowadays, it is still very difficult for Android users to transfer application data from one Android device to another. 
If the Android device is rooted, it is very easy to migrate your contacts, call logs, SMS messages, Wi-Fi, application data, and data under ``/sdcard`` to another device. 
Furthermore, after migrating data using root privileges, there is no need to log in to each internet application one by one, which is the convenience we have been hoping for.

## 原因

本文档将简要讨论执行 LRFP 的一些原因。

### 为什么要 root

即使全民 root 和一键 root 的时代已经过去了十年，一些用户依旧在使用他们的安卓手机前先执行 root。
目前，大部分人认为安卓手机系统已经足够完善，没有 root 的需要。
但是，即使是当下，安卓用户仍难以将应用程序数据从一台安卓设备传输到另一台安卓设备。
如果安卓设备已 root，则很容易将联系人、通话记录、短信记录、WiFi、应用数据和 ``/sdcard`` 下的数据迁移到另一台设备。
甚至，使用 root 权限迁移数据后，不需要再逐个登录互联网应用，这是我们所期望的便利。
